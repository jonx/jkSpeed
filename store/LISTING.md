## Detailed Description
Control the speed of any HTML5 video on any page — including YouTube Shorts and videos embedded in iframes.

Features:
- Detects all videos on the current page, including Shorts and iframes
- Speed presets: 1x and 1.5x
- Fine-tune speed with +/- 0.25x buttons
- Play/pause toggle
- Real Picture-in-Picture — pops the video into a native OS window, unlike YouTube's built-in "miniplayer" that stays trapped inside the page. Works on Shorts too!
- Customizable keyboard shortcuts (default: S slow down, D speed up) — overrides page shortcuts, fixing issues with YouTube shortcuts not working on non-US keyboards (e.g. French AZERTY)
- Speed overlay — shows current speed on the video when using shortcuts
- Remember speed per site — opt-in setting, hostnames are hashed for privacy
- Extension icon lights up when a video is detected
- Automatically finds new videos as they load

How to use:
1. Navigate to any page with a video
2. Click the jkSpeed icon (it lights up when videos are found)
3. Use the controls to adjust speed, play/pause, or enter PiP mode

Permissions explained:
- activeTab: Access the current tab only when you click the extension
- webNavigation: Find videos inside embedded iframes
- storage: Save your shortcut preferences and optional site speed settings locally

No data is collected or transmitted. Site hostnames are salted and hashed if you enable "remember speed" — can't be easily reversed. Fully open source.
